# AI Setting Configuration

### Step 1: AI Settings
1. Log into your Admin Panel and navigate to the `AI Setup` section.
2. Click on the `AI Settings` tab.
<Image src={'/images/admin/ai/config/1.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />

#### Open AI Settings
1. Change the OpenAI Default Model to the model you want to use. br
    For example, I want to use the `GPT-3.5 Turbo` model, so I will change the OpenAI Default Model to `GPT-3.5 Turbo`. <br/>
    Note: Your subscription must support the model you select.
2. Click the `Update Changes` button.
<Image src={'/images/admin/ai/config/2.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />


#### Stability AI Settings
1. You can change the default Random API key to a specific API key you want to use.
2. Click the `Update Changes` button. <br/>
    For example, I want to use Random API.
<Image src={'/images/admin/ai/config/3.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />

#### Rapid API Settings
1. You can change the default Random API key to a specific API key you want to use.
2. Click the `Update Changes` button. <br/>
    For example, I want to use Random API.
