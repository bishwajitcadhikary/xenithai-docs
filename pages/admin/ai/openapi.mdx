# OpenAI API

### Step 1: Go to Open AI API Settings Page
1. Log into your Admin Panel and navigate to the `AI Setup` section.
<Image src={'/images/admin/ai/openai/1.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />
2. Click on the `OpenAI API` tab.
<Image src={'/images/admin/ai/openai/2.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />
3. Click on the `Add API Key` button.
<Image src={'/images/admin/ai/openai/3.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />

### Step 1: Register for an Open AI API Key
Open your browser and navigate to [https://beta.openai.com/](https://beta.openai.com/).
Click on the `Sign up` button in the top right corner.
Fill out the form and click `Sign up`.
You will receive an email with a link to verify your email address.
Click on the link in the email to verify your email address. You will be redirected to the Open AI dashboard.

### Step 2: Generate an API Key
1. Click on the Profile icon in the top right corner and select `View API Keys` or click [here](https://platform.openai.com/account/api-keys).
<Image src={'/images/admin/ai/openai/4.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />
2. Click on the `Create new secret key` button.
<Image src={'/images/admin/ai/openai/5.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />
3. Give a name to your API key and click on the `Create new secret key` button.  You will be presented with your API key. Copy the API key and save it somewhere safe. You will need it later.
<Image src={'/images/admin/ai/openai/6.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />
4. Copy the API key and save it somewhere safe. You will need it later.
<Image src={'/images/admin/ai/openai/7.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />

### Step 3: Configure the Key
1. Go back to your Admin Panel and paste the API key in the `API Key` field.
2. Click on the `Save` button.
<Image src={'/images/admin/ai/openai/8.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />
3. You will see a success message if the API key was saved successfully.
<Image src={'/images/admin/ai/openai/9.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />
