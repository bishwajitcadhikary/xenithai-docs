# Currencies

Here you can see the currencies that supported your application.

### Step 1: Go to Currencies Page
1. Log into your Admin Panel and navigate to the `Business Setup` section.
2. Click on the `Currencies` tab.
<Image src={'/images/admin/business/currency/1.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />

### List
Here you can see all the currencies that are available in your application. You can also add new currencies, edit existing ones and delete them.
<Image src={'/images/admin/business/currency/2.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />

#### Configure Exchange Rates API
1. To configure the Exchange Rates API, go to [https://yourdomain.com/admin/business/currencies/settings](https://yourdomain.com/admin/business/currencies/settings). <br/>
Note: Also you can set auto sync option to sync currencies automatically. <br/>
2. Enter your API key and click on the `Update Changes` button.
<Image src={'/images/admin/business/currency/3.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />
3. To get a api key you need to create an account on [https://apilayer.com/](https://apilayer.com/).
4. After you have created an account you can get the api key from [https://apilayer.com/marketplace/exchangerates_data-api](https://apilayer.com/marketplace/exchangerates_data-api). <br/>
5. Subscribe to the `Exchange Rates API` and get your API key.
6. Copy the API key and paste it in the `API Key` field.
<Image src={'/images/admin/business/currency/4.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />
<br/>
<Image src={'/images/admin/business/currency/5.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />

### Update Exchange Rates
1. Go back to the `Currencies` page.
2. Click on the refresh icon button.
