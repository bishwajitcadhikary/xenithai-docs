# Users

### Step 1: Go to User Page
1. Log into your Admin Panel and navigate to the "Users" section.
<Image src={"/images/admin/users/1.png"} width={1000} height={500} className={"bg-img"} alt={"image"}/>

### Step 2: Create User
1. Click on the "Create User" button.
<Image src={"/images/admin/users/2.png"} width={1000} height={500} className={"bg-img"} alt={"image"}/>
2. Enter the user details.
3. Click on the "Save" button.
<Image src={"/images/admin/users/3.png"} width={1000} height={500} className={"bg-img"} alt={"image"}/>
<br/>
<Image src={"/images/admin/users/3-1.png"} width={1000} height={500} className={"bg-img"} alt={"image"}/>

### Step 3: Edit User
1. Click on the "Edit" button to edit the user.
<Image src={"/images/admin/users/4.png"} width={1000} height={500} className={"bg-img"} alt={"image"}/>
2. Edit the user details.
3. Click on the "Update Changes" button.
<Image src={"/images/admin/users/5.png"} width={1000} height={500} className={"bg-img"} alt={"image"}/>

### Step 4: Delete User
1. Click on the "Delete" button to delete the user.
<Image src={"/images/admin/users/6.png"} width={1000} height={500} className={"bg-img"} alt={"image"}/>

### Step 5: Bulk Delete Users
1. Select the users you want to delete or click on the "Select All" checkbox to select all the users.
