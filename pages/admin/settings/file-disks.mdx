# File Disks

### Step 1: Go to File Disks Page
1. Log into your Admin Panel and navigate to the `Settings` section.
2. Click on the `General` tab.
<Image src={'/images/admin/settings/file-disks/1.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />

### Step 2: Enable or Disable File Disks
1. Update file disks settings as per your requirements.
2. Click on the `Save Changes` button to save the changes.
<Image src={'/images/admin/settings/file-disks/2.png'} className={'bg-img'} width={1000} height={500} alt={'Image'} />

### Step 3: Fill in the File Disk Information
This file system is used to store all the files uploaded by the users. You can use the default file system, or you can use a cloud storage service like Amazon S3.

Supported File Systems: `Amazon S3`, `Wasabi`, `Contabo`, `FTP, Local`
